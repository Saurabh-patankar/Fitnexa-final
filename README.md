# 🚀 FitNexa – AI-Powered Smart Fitness Platform

## 💡 Overview

**FitNexa** is a full-stack AI-powered fitness web application designed to enhance user engagement, track fitness activities, and provide intelligent guidance. It combines real-time tracking, computer vision, and AI-driven insights to deliver a personalized fitness experience.

This project demonstrates strong capabilities in **frontend development, backend integration, real-time systems, and applied AI**.

---

## ✨ Key Features

### 🏋️ Smart Workout Timer

* Real-time workout and rest tracking
* Voice-controlled commands (Start, Stop, Reset)
* Helps users maintain structured workout sessions

---

### 📊 Nutrition & Macro Tracker

* Track daily intake of protein, carbs, and fats
* AI-based meal suggestions
* Visual progress tracking with charts

---

### 🤖 AI Fitness Advisor

* Chat-based assistant for workout and diet planning
* Provides personalized suggestions
* Enhances user engagement with conversational UI

---

### 👥 Community Platform

* Post updates, interact with other users
* Like and comment system
* Encourages motivation and social engagement

---

### 📷 QR-Based Gym Check-In

* Contactless attendance system using QR codes
* Tracks user check-in/check-out
* Maintains attendance history and streaks

---

### 🛡️ Admin Dashboard

* Monitor all user activity
* Track gym attendance in real-time
* Analyze user engagement and system usage

---

### 🧠 AI Form Corrector (🔥 Highlight Feature)

* Real-time posture detection using webcam/video
* Detects body keypoints and calculates joint angles
* Provides feedback on exercise form
* Counts repetitions automatically

## 🧠 How It Works (Core Logic)

1. User interacts with UI (timer, nutrition, AI chat, etc.)
2. Data is processed via frontend + backend APIs
3. AI modules:

   * Pose detection extracts body landmarks
   * Joint angles are calculated
   * Movement patterns are analyzed
4. Results are displayed in real-time

```bash

# Navigate to project

npm run dev

# Run backend
npm run server
```
